## Irrlicht-ce
Supporting real-time vedio transfer.

### Build
1. Ubuntu 20 LTS
```bash
sudo apt install libxxf86vm-dev
sudo apt install libgl1-mesa-dev libglu1-mesa-dev
```
2. MacOSX
Using Xcode follow readme.txt is okay.
